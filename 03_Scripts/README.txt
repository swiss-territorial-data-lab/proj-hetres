Script in developments for the different tracks and tools explored. 

- colorisation : colorisation of point cloud by help or images second https://data.grandlyon.com/actualites/traitement-du-nuage-de-points-avec-des-outils-libres
- DFT : Some custom batches to use with Digital Forestry Toolbox
- DL : Application of ResNet for synthetic images of the beech trees. 
- FHI : Scripts replicating the methodology of the article https://www.tandfonline.com/doi/full/10.1080/17538947.2022.2059114 and related analysis. 
- potree : HTML structure for potree viewer 
- treedet-toolkit : some custom adaptation of https://github.com/swiss-territorial-data-lab/treedet-toolkit for the purpose of the projet and additional analysis. Very specific, not meant to be integrate in the orignal repository of treedet-toolkit.
